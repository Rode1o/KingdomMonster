# KingdomMonster

Developed with Unreal Engine 5

MovementCharacter was based on New Enhanced Input Plugin:

<p> W : FORWARD </p>
<p> A : LEFT </p>
<p> S : BACKWARD </p>
<p> D : RIGHT </p>
<p> SPACE : JUMP </p>
<p> LEFTCLICK : ATTACK </p>
<p> SHIFTLEFT : SPRINT </p>
